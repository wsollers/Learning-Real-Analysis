\noindent
\textbf{Source.}
\srccite{TaoAnalysis1}{Chapter~2, \S2.3, Corollary~2.3.7}.

\vspace{0.75em}

\noindent
\textbf{Goal.}
Prove Corollary~2.3.7 (Cancellation law for multiplication): if $a, b, c$
are natural numbers with $ac = bc$ and $c \neq 0$, then $a = b$.

\medskip
\noindent
\textbf{Logical form.}
$\forall\, a, b, c \in \mathbb{N},\quad ac = bc \text{ and } c \neq 0
\;\Rightarrow\; a = b.$

\vspace{0.75em}

\noindent\textbf{Available toolkit.}
P1--P5, Proposition~2.2.12, Proposition~2.2.13 (trichotomy),
Proposition~2.3.6 (multiplication preserves order).

\vspace{0.75em}

\noindent
\textbf{Strategy.}
(Fill in before beginning the proof. Hint: use trichotomy on $a$ and $b$,
then derive a contradiction in each non-equal case using P2.3.6.)
