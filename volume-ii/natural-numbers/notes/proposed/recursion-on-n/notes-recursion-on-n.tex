\section{Recursion on \texorpdfstring{$\mathbb{N}$}{N}}

This section is the engine room of the chapter. It should introduce the
recursion problem, state the recursion theorem, and explain how initial value
plus successor rule produces well-defined functions on $\mathbb{N}$.

\begin{remark}[Why this comes early]
Addition, multiplication, exponentiation, and many canonical sequences on
$\mathbb{N}$ are all built recursively. The chapter should earn those
operations here before using them later.
\end{remark}
