\section{Order on \texorpdfstring{$\mathbb{N}$}{N}}

This section should define \(m \le n\) by the existence of an additive
difference and then build the full order toolkit: reflexivity, antisymmetry,
transitivity, trichotomy, and compatibility with addition and multiplication.

\begin{remark}[Why this precedes structural packaging]
Order on $\mathbb{N}$ is concrete arithmetic structure, not a later abstract
summary. It deserves to be developed before the chapter steps back to name
the overall algebraic object.
\end{remark}
