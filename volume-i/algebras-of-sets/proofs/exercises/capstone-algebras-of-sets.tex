\label{prf:capstone-algebras-of-sets}

\begin{proof}
The claim is that $(\mathcal{P}(X), \mathbin{\triangle}, \cap)$
is a Boolean ring: a commutative ring with unity in which every
element satisfies $A \cap A = A$.

Specifically, we must verify:
\begin{enumerate}
  \item $(\mathcal{P}(X), \triangle)$ is an abelian group with
        identity $\emptyset$.
  \item $(\mathcal{P}(X), \cap)$ is a commutative monoid with
        identity $X$.
  \item $\cap$ distributes over $\triangle$.
  \item Every element is idempotent under $\cap$.
\end{enumerate}
