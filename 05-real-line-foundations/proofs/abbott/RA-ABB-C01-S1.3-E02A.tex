\noindent
\textbf{Goal.} Give an example of a set $B$ with $\inf B \ge \sup B$, or state that the request is impossible.

\vspace{0.75em}

\noindent
\textbf{Proof (stub).}
\begin{itemize}
\item Decide first whether the request is possible. If impossible, prove impossibility from definitions.
\item If possible, propose a candidate $B$ and compute/justify $\inf B$ and $\sup B$.
\item Use: for any nonempty bounded set, $\inf B \le \sup B$. If $B$ is empty or unbounded, note what fails.
\end{itemize}
