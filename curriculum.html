Implementation components

Problem structure
  • Two-player stochastic differential game — pursuer minimizes distance to evader; evader is Brownian (no control) on (𝕋², g(t))
  • Value function — V(t,x,y) = expected cost under optimal pursuit; satisfies stochastic HJB on 𝕋² × 𝕋²

  • Evolving metric — g(t) changes geodesic distances at each moment; cost function is time-varying
  • Open problem — the combination of DDE pursuit, Brownian evader, and evolving Riemannian metric on 𝕋² is not fully treated in the literature
