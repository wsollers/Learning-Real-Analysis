\textbf{Exercise.}
A function $f \colon \mathbf{R} \to \mathbf{R}$ is called \emph{even} if
\[
  f(-x) = f(x)
\]
for all $x \in \mathbf{R}$.
A function $f \colon \mathbf{R} \to \mathbf{R}$ is called \emph{odd} if
\[
  f(-x) = -f(x)
\]
for all $x \in \mathbf{R}$.
Let $V_e$ denote the set of real-valued even functions on $\mathbf{R}$
and let $V_o$ denote the set of real-valued odd functions on $\mathbf{R}$.
Show that $\mathbf{R}^{\mathbf{R}} = V_e \oplus V_o$.
