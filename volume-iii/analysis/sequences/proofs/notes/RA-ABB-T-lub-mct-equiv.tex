\bigskip

\noindent
\textbf{Source.} See \texttt{monotone-approximation.tex}.

\vspace{0.75em}

\noindent
\textbf{Goal.} Over an ordered field, the Least Upper Bound Property is equivalent to the Monotone Convergence Theorem.

\vspace{0.75em}

\noindent
\textbf{Logical form.}
\[
  \text{LUB holds} \Longleftrightarrow \text{MCT holds}.
\]

\vspace{0.75em}

\noindent
\textbf{Proof strategy.} (LUB $\Rightarrow$ MCT) is the standard proof: use sup. (MCT $\Rightarrow$ LUB) is more subtle: given a bounded nonempty set $S$, build a monotone sequence converging to sup $S$ using MCT.
